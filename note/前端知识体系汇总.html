                                    前端知识体系汇总

                                    keepview 2019-12-01 10:00:20

                                    # 基础

                                    HTML

                                    用于创建网页的标准标记语言

                                    CSS

                                    层叠样式表

                                    JavaScript

                                    具有函数优先的轻量级，解释型或即时编译型的编程语言

                                    # CSS 预处理器

                                    Less

                                    一门向后兼容的 CSS 扩展语言

                                    Sass

                                    最成熟、稳定和强大的专业级 CSS 扩展语言

                                    Stylus

                                    富于表现力、动态的、健壮的 CSS 扩展语言

                                    # 模块化开发

                                    HTML5 Boilerplate

                                    网上最受欢迎的前端模板

                                    Normalize.css

                                    在默认的HTML元素样式上提供了跨浏览器的高度一致性

                                    OOCSS

                                    面向对象的CSS

                                    CssReset

                                    重置浏览器默认样式

                                    # UI 组件库

                                    Web Components

                                    浏览器的原生组件

                                    Bootstrap

                                    最受欢迎的 HTML、CSS 和 JS 框架, 用于开发响应式布局、移动设备优先的 WEB 项目

                                    Element UI

                                    基于 Vue 2.0 的桌面端组件库

                                    # 数据可视化

                                    Web 图形开发

                                    绘制出动态图像，或者是对图像进行巧妙处理

                                    Canvas

                                    通过JavaScript 和 HTML的 canvas 元素来绘制图形的方式

                                    WebGL

                                    一种JavaScript API，用于在任何兼容的Web浏览器中呈现交互式3D和2D图形，而无需使用插件

                                    # 浏览器

                                    Web API 接口参考

                                    在使用 JavaScript 编写 Web 代码时，有许多 Web API 可供调用

                                    浏览器扩展

                                    Extensions「扩展」 可以扩展和修改一个浏览器的能力

                                    # 代码规范

                                    编写可维护的CSS

                                    SF : 参与大型团队开发项目时，遵守一致的规则极为重要

                                    CSS Lint

                                    NEC : 更好的 CSS 样式解决方案

                                    # 网络

                                    HTTP

                                    用于传输超媒体文档（例如 HTML）的应用层协议

                                    TCP/IP详解卷1: 协议

                                    一本详细的TCP/IP协议指南

                                    # Web 服务

                                    Node.js

                                    一个基于 Chrome V8 引擎 的 JavaScript 运行时

                                    Nginx

                                    一个高性能的HTTP和反向代理web服务器，同时也提供了IMAP/POP3/SMTP服务

                                    Express

                                    基于 Node.js 平台，快速、开放、极简的 Web 开发框架

                                    Koa

                                    基于 Node.js 平台的下一代 web 开发框架

                                    ThinkJS

                                    一款面向未来开发的 Node.js 框架

                                    Egg

                                    为企业级框架和应用而生

                                    # Web 安全

                                    XSS

                                    跨站脚本攻击

                                    CSRF

                                    跨站点伪造请求攻击

                                    跨 iframe 攻击

                                    网页中被插入恶意 iframe 造成攻击

                                    Clickjacking 安全漏洞

                                    当你访问一个恶意网站的时候，攻击者可以控制你的浏览器对一些链接的访问

                                    # 构建工具/包管理器

                                    webpack

                                    JavaScript 应用程序的静态模块打包器

                                    Grunt

                                    JavaScript 世界的构建工具

                                    Bower

                                    依赖包（前端）管理工具

                                    NPM

                                    世界上最大的软件注册表（依赖包）

                                    Yarn

                                    快速、可靠、安全的依赖管理工具

                                    # 调试工具

                                    EOLINKER

                                    API在线管理平台

                                    Mock.js

                                    生成随机数据，拦截 Ajax 请求

                                    Chrome

                                    谷歌浏览器开发调试扩展工具，谷歌商店安装，需要翻墙

                                    Firebug

                                    火狐浏览器开发调试扩展工具

                                    HTTPWatch

                                    网页数据分析工具.集成在Internet Explorer工具栏

                                    Fiddler

                                    http协议调试代理工具

                                    Weinre

                                    移动端开发调试工具

                                    # 开发工具

                                    VS Code

                                    用于编写现代 Web 和云应用的跨平台源代码编辑器

                                    Sublime Text

                                    用于代码、标记和散文的精致文本编辑器

                                    # 设计工具

                                    PxCook

                                    设计图自动标注工具

                                    Mockplus

                                    简洁高效的原型设计工具

                                    iconfont

                                    阿里巴巴矢量图标库

                                    # 文档工具

                                    VuePress

                                    Vue 驱动的静态站点生成工具

                                    语雀 - 云端知识库

                                    提供与众不同的知识管理，打造轻松流畅的工作协同

                                    XMind

                                    思维导图

                                    # 主流框架&生态

                                    Vue 生态

                                    Vue.js

                                    渐进式 JavaScript 框架

                                    Vue-Cli

                                    Vue.js 开发的标准工具

                                    Vue Router

                                    Vue.js 官方的路由管理器

                                    Vuex

                                    Vue.js 开发的状态管理模式

                                    React 生态

                                    React.js

                                    用于构建用户界面的 JavaScript 库

                                    Redux

                                    JavaScript 状态容器，提供可预测化的状态管理；除了和 React 一起用外，还支持其它界面库

                                    React Router

                                    React.js 官方的路由管理器

                                    Angular 生态

                                    Angular.js

                                    一款优秀的前端JS框架

                                    jQuery 生态

                                    jQuery

                                    一个 JavaScript 库，极大地简化了 JavaScript 编程

                                    jQuery Mobile

                                    创建移动 web 应用程序的框架

                                    Zepto

                                    一个轻量级的针对现代高级浏览器的JavaScript库

                                    # 常用插件&依赖库

                                    Axios

                                    基于 promise 的 HTTP 库

                                    Hammer.js

                                    轻量级的触屏设备手势库

                                    Swiper

                                    移动端触摸滑动插件

                                    Highlight.js

                                    代码高亮

                                    Chart.js

                                    基于 HTML5 的 JavaScript 图表库

                                    Echarts

                                    使用 JavaScript 实现的开源可视化图表库

                                    D3

                                    Particles.js

                                    用于创建粒子的轻量级 JavaScript 库

                                    slidePage

                                    轻量无依赖的全屏滚动组件

                                    resize-detector

                                    监听元素大小变化

                                    Typed.js

                                    Instantclick

                                    鼠标 hover 超链接时预加载资源

                                    svg-sprite-loader

                                    Animate.css

                                    预设 CSS3 动画库

                                    Anime.js

                                    Shepherd.js

                                    mavon-editor

                                    TinyEditor

                                    WangEditor
